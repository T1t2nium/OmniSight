# Visual Hierarchy: Before & After Examples

## Example 1: Marketing Landing Page

### Before
```
Logo:          72px (dominates)
Headline:      24px, weight 400
CTA Button:    14px, text link, grey
Body:          16px
```
**Problems:**
- Logo draws more attention than value proposition
- CTA is weak (text link, small)
- No clear visual path

### After
```
Logo:          32px (appropriately sized)
Headline:      48px, weight 700
CTA Button:    16px, solid brand color
Body:          18px, weight 400, grey
```
**Improvements:**
- Clear hierarchy: Headline → CTA → Body
- CTA stands out with brand color
- Logo present but not dominant

---

## Example 2: Dashboard Card

### Before
```
Card Title:    20px, weight 700, dark
Metric Value:  16px, weight 400, grey
Action Link:   12px, text only
```
**Problems:**
- Title dominates over the actual data
- Metric (most important) is visually weak
- Action is too small to notice

### After
```
Metric Value:  32px, weight 700, dark  ← Primary
Card Title:    14px, weight 500, grey  ← Secondary
Action Link:   14px, brand color       ← Tertiary
```
**Improvements:**
- Metric is the hero
- Title supports without competing
- Action is discoverable

---

## Example 3: Form Layout

### Before
```
Section Title: 24px, weight 700
Label:         16px, weight 600
Input:         16px
Button:        16px, outline
```
**Problems:**
- Section title competes with form content
- Label uses weight instead of position
- Button doesn't look primary

### After
```
Section Title: 18px, weight 500, grey  ← De-emphasized
Label:         14px, weight 400, grey  ← Subtle
Input:         16px
Button:        16px, solid brand color ← Primary
```
**Improvements:**
- Title is supportive, not dominant
- Label proximity creates relationship
- Button is clearly the primary action
